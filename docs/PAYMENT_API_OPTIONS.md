# 💰 무료 송금/결제 API 옵션 가이드

## 📋 개요

한국에서 무료로 사용할 수 있는 송금/결제 API 옵션들을 정리한 문서입니다.

## 🆓 무료 티어가 있는 결제 API

### 1. **토스페이먼츠 (Toss Payments)**

**특징:**

- 한국 대표 결제 서비스
- 개발자 친화적인 API
- 무료 티어 제공 (제한적)

**무료 조건:**

- 개발/테스트 환경에서 무료
- 실제 결제는 수수료 발생 (일반적으로 2.5~3.5%)
- 월 거래액 기준으로 수수료 차등 적용

**API 문서:**

- https://developers.toss.im/

**장점:**

- 한국어 문서 완비
- 다양한 결제 수단 지원 (카드, 계좌이체, 가상계좌 등)
- 모바일/웹 모두 지원

**단점:**

- 실제 송금은 수수료 발생
- 사업자 등록 필요

---

### 2. **아임포트 (Iamport)**

**특징:**

- 국내 대표 PG 연동 서비스
- 다양한 PG사 통합 지원

**무료 조건:**

- 개발/테스트 환경 무료
- 실제 결제는 각 PG사 수수료 적용
- 월 구독료 없음 (수수료만)

**API 문서:**

- https://developers.iamport.kr/

**장점:**

- 여러 PG사 통합 관리
- 무료 티어 제공
- 한국어 지원

**단점:**

- 실제 송금은 수수료 발생
- PG사별 수수료 차이

---

### 3. **KG이니시스 (KG Inicis)**

**특징:**

- 국내 주요 PG사
- 안정적인 서비스

**무료 조건:**

- 개발/테스트 환경 무료
- 실제 결제는 수수료 발생

**API 문서:**

- https://www.kginicis.com/

**장점:**

- 안정적인 서비스
- 다양한 결제 수단

**단점:**

- 사업자 등록 필요
- 수수료 발생

## 🏦 오픈뱅킹 API (공공 API)

### **금융결제원 오픈뱅킹 API**

**특징:**

- 한국은행과 금융결제원이 제공하는 공공 API
- 은행 간 계좌이체 지원
- **무료** (API 사용료 없음)

**제공 기능:**

- 계좌조회
- 잔액조회
- 계좌이체 (송금)
- 거래내역조회

**무료 조건:**

- API 사용료 무료
- 단, 각 은행의 계좌이체 수수료는 별도 (일반적으로 무료 또는 저렴)

**API 문서:**

- https://openbanking.or.kr/
- https://www.openbanking.or.kr/apiservice

**장점:**

- ✅ **완전 무료** (API 사용료)
- 공공 API로 안정적
- 대부분의 은행 지원
- 실제 송금 기능 제공

**단점:**

- 사용자 인증 절차 복잡 (OAuth 2.0)
- 각 은행별로 API 연동 필요
- 개발 복잡도 높음

**신청 방법:**

1. 금융결제원 오픈뱅킹 포털 가입
2. 앱 등록 및 인증
3. 각 은행별 API 신청

## 🌐 해외 송금 API

### 1. **Wise (구 TransferWise) API**

**특징:**

- 국제 송금 전문
- API 제공

**무료 조건:**

- API 사용료는 무료일 수 있으나
- 실제 송금 수수료 발생 (송금액의 0.35~2% 정도)

**API 문서:**

- https://wise.com/api

**장점:**

- 해외 송금 전문
- 환율 우대

**단점:**

- 국내 송금에는 부적합
- 수수료 발생

---

### 2. **Revolut API**

**특징:**

- 글로벌 핀테크 서비스
- 기업용 API 제공

**무료 조건:**

- 무료 플랜 존재
- 제한된 기능

**API 문서:**

- https://www.revolut.com/business/api

**장점:**

- 무료 플랜 제공
- 다양한 기능

**단점:**

- 한국 서비스 제한적
- 기업 고객 대상

## 🎯 추천 사항

### **국내 송금이 목적이라면:**

1. **오픈뱅킹 API** (최우선 추천)

   - ✅ 완전 무료 (API 사용료)
   - ✅ 실제 송금 기능
   - ✅ 공공 API로 안정적
   - ⚠️ 개발 복잡도 높음

2. **토스페이먼츠**

   - ✅ 개발이 쉬움
   - ✅ 한국어 문서 완비
   - ⚠️ 수수료 발생 (2.5~3.5%)

3. **아임포트**
   - ✅ 여러 PG사 통합
   - ✅ 무료 티어
   - ⚠️ 수수료 발생

### **해외 송금이 목적이라면:**

1. **Wise API**
2. **Revolut API**

## 📝 참고사항

1. **사업자 등록 필요**

   - 대부분의 결제 API는 사업자 등록이 필요합니다
   - 오픈뱅킹 API도 앱 등록 시 사업자 정보 필요

2. **보안 및 규제**

   - 금융 거래는 엄격한 보안 요구사항이 있습니다
   - PCI-DSS 준수 필요
   - 개인정보보호법 준수

3. **테스트 환경**

   - 대부분의 서비스는 테스트/개발 환경을 무료로 제공합니다
   - 실제 운영 전 충분한 테스트 권장

4. **수수료 구조**
   - API 사용료와 실제 송금 수수료는 별개입니다
   - 오픈뱅킹은 API 사용료 무료, 송금 수수료는 은행별 상이
